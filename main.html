Question-3: What is event delegation in the DOM?
Event delegation in the DOM is a technique where a single event listener is attached to a parent element to manage events for its child elements, even if those children are dynamically added later. Instead of attaching event listeners to multiple child elements individually, the parent element listens for events, taking advantage of event propagation (bubbling).
Question-4: What is event bubbling in the DOM?
Event bubbling in the DOM is a mechanism where an event starts at the element where it occurs and propagates (or "bubbles up") through its ancestors in the DOM hierarchy, all the way to the document root.
